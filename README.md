Question1 Answer
```
select s.name, snumber, sdepartment, p.name 
from student s, professor p 
where s.pnumber = p.pnumber(+);
```

Question2 Answer
```
select s.name, p.name, s.pnumber 
from student s, professor p
where s.pnumber = p.pnumber(+);
```
